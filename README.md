# Water Learn

Water Learn adalah aplikasi Android yang dikembangkan untuk mendukung **Sustainable Development Goals (SDGs) Nomor 6: Air Bersih dan Sanitasi Layak**. Aplikasi ini bertujuan memberikan edukasi kepada masyarakat mengenai pentingnya air bersih, sanitasi yang layak, serta pengelolaan sumber daya air yang berkelanjutan melalui berbagai fitur edukatif dan interaktif.

## Fitur Utama

- Login dan Registrasi Pengguna
- Grafik Penggunaan Air dan Sanitasi
- Video Edukasi
- Artikel Edukasi
- Chatbot Interaktif
- Manajemen Data Kontak (CRUD)
- Dashboard

## Tujuan Aplikasi

Water Learn dikembangkan untuk:

- Meningkatkan kesadaran masyarakat mengenai pentingnya air bersih dan sanitasi yang layak.
- Memberikan edukasi mengenai pengelolaan sumber daya air yang berkelanjutan.
- Membantu pengguna memahami dampak perubahan iklim terhadap ketersediaan air bersih.
- Mendukung pencapaian SDGs Nomor 6 tentang Air Bersih dan Sanitasi Layak.

## Teknologi yang Digunakan

- Java
- Android Studio
- Firebase Authentication
- Firebase Realtime Database
- Firebase Storage
- OkHttp
- JSON API

## Fitur Aplikasi

### Artikel dan Video Edukasi

Menyediakan informasi dan video edukasi mengenai pentingnya air bersih dan sanitasi yang layak.

### Chatbot Interaktif

Memungkinkan pengguna mengajukan pertanyaan dan memperoleh jawaban secara langsung melalui chatbot.

### Visualisasi Data

Menampilkan grafik terkait penggunaan air, konsumsi air minum, dan sanitasi untuk membantu pengguna memahami kondisi serta pentingnya pengelolaan air.

### Manajemen Data Kontak

Mendukung operasi Create, Read, Update, dan Delete (CRUD) yang terhubung dengan Firebase Realtime Database.

## Instalasi

1. Clone repository:

```bash
git clone https://github.com/ys124xkd/WaterLearn.git
```

2. Buka proyek menggunakan Android Studio.
3. Lakukan sinkronisasi Gradle.
4. Konfigurasikan layanan Firebase sesuai kebutuhan.
5. Jalankan aplikasi pada emulator atau perangkat Android.
